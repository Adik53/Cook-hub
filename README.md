**Cook Hub** is a full-featured social network for cooking enthusiasts. Users can search for recipes by ingredients or name, create and share their own recipes, follow other cooks, like/dislike recipes, leave comments, and enjoy a personalized feed.

The app supports **4 languages**, features **light and dark themes**, and has a clean, responsive design built with modern technologies.
